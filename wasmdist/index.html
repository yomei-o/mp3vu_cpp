🎚️ MP3 Player + VU / LED メーター

MP3のデコード（minimp3）・音量レベル計算・メーターの描画（VUの針・目盛り・レッドゾーン／LEDスペクトラム）まで全部 C/C++ → WASM。 JSはブラウザ必須の部分だけ（ファイル選択・Web Audio再生・canvas転送）。

--:-- / --:--

MP3ファイルを開いてください（ドラッグ&ドロップも可・端末内で完結し送信しません）。

メーターは C++ が RGBA バッファへ毎フレーム描画し canvas へ転送しています。
VU：RMS＋VU的な針の弾道（アタック速め・戻り遅め）、ピークホールド＋クリップLED付き。 LED：自作FFTで対数帯域に分けたスペクトラムを緑/黄/赤のLED段で表示（各バンドにピーク保持ドット）。 ボタンで切替できます。
ソース: github.com/yomei-o/mp3vu_cpp（minimp3 は public domain）
